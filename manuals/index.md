# 프로젝트 매뉴얼 목차 (Manual Index)

이 파일은 전체 매뉴얼의 목차입니다. AI는 작업 시작 전 이 파일을 먼저 읽고,
관련된 챕터만 선택적으로 로드합니다.

## 챕터 목록

| 챕터 | 파일 | 키워드 | 설명 |
|------|------|--------|------|
| 1. 아키텍처 | `chapters/architecture.md` | 구조, 설계, 폴더, 레이어 | 프로젝트 구조 및 레이어 분리 규칙 |
| 2. 프론트엔드 | `chapters/frontend.md` | UI, 컴포넌트, 스타일, React | 프론트엔드 개발 표준 |
| 3. 백엔드 | `chapters/backend.md` | API, 서버, 라우트, 인증 | 백엔드 개발 표준 |
| 4. 데이터베이스 | `chapters/database.md` | DB, 모델, 스키마, 마이그레이션 | 데이터베이스 설계 및 운용 규칙 |
| 5. 보안 | `chapters/security.md` | 인증, 권한, 토큰, 암호화 | 보안 가이드라인 |

## 매뉴얼 사용법

1. **키워드 매칭**: 사용자 요청에 위 키워드가 포함되면 해당 챕터를 자동 로드
2. **독립적 읽기**: 각 챕터는 독립적으로 읽어도 이해 가능하도록 작성됨
3. **우선순위**: 매뉴얼 규칙 > AI 판단 > 일반적 관행
