# 프로젝트 상태 확인

현재 진행 상태를 요약합니다.

## Step 1: 3대 문서 읽기
- `docs/PLAN.md` → 전체 목표 및 설계
- `docs/TODO.md` → 진행률 (완료/대기/블로커)
- `docs/CONTEXT.md` → 기술적 결정 이력

## Step 2: 상태 요약 출력

```markdown
## 프로젝트 상태

**목표**: (PLAN.md에서)
**진행률**: X/Y 완료 (Z%)

### ✅ 완료
- (완료된 태스크)

### 🔄 진행 중
- (진행 중인 태스크)

### ⬜ 대기
- (아직 시작 안 한 태스크)

### 🚫 블로커
- (막혀있는 것)

### 최근 결정 사항
- (CONTEXT.md에서 최근 항목)
```
